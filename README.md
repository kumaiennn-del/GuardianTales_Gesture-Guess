# Gesture Guess Game

## 项目简介

这是一个基于 Python 开发的小游戏，名为 **Gesture Guess**，玩家通过模仿手势来猜测相应的动作或物体。游戏使用了 Python 语言和一些基本的游戏开发库，旨在提供一个有趣且互动性强的用户体验。

## 游戏功能

- **得分系统**：控制者可以根据玩家答对与否加减分数。
- **音效和背景音乐**：游戏中加入了背景音乐和音效，使游戏体验更具互动性。
- **关卡系统**：坎公游戏内的角色和道具会在每关中随机出现。

## 安装与运行

### 环境要求
- Python 3.x
- 游戏需要安装 `pygame` 库进行图形和音效处理。

### 安装步骤

1. **安装 Python**：
   确保你已经安装了 Python 3.x。你可以在 [Python 官网](https://www.python.org/) 下载并安装。

2. **安装所需库**：
   游戏依赖于 `pygame` 库来实现图形和音效。你可以使用以下命令安装所需的依赖：
   ```bash
   pip install pygame
   ```
   
3. **下载项目文件**：
将项目文件下载到本地，或者通过 Git 克隆仓库：

git clone https://github.com/your-username/GestureGuessGame.git

4. **运行游戏**：
在终端或命令行中，进入项目目录并运行游戏主文件：
   ```bash
   python Gesture&Guess.py
   ```
